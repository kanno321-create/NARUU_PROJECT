## 3. 기능별 상세 분석

### 3.1 기초자료등록 (Master Data Management)

#### 3.1.1 자사정보등록
- **목적**: 회사 기본 정보 등록 (사업자등록번호, 대표자, 주소 등)
- **필드**: 회사명, 사업자번호, 대표자, 업태, 종목, 전화, 팩스, 주소, 이메일
- **연결고리**: 모든 문서(견적서, 세금계산서 등)의 발신자 정보로 사용

#### 3.1.2 거래처등록
- **목적**: 고객사 및 공급업체 정보 관리
- **필드**: 거래처코드, 상호, 사업자번호, 대표자, 담당자, 연락처, 주소, 거래구분(매출/매입), 신용한도, 담당사원
- **연결고리**:
  - → 전표작성 (매출/매입 시 거래처 선택)
  - → 채권채무 (거래처별 외상 잔액)
  - → 세금계산서 (수신자 정보)

#### 3.1.3 상품등록
- **목적**: 판매/구매 상품 정보 관리
- **필드**: 상품코드, 상품명, 규격, 단위, 분류, 매입단가, 매출단가, 재고수량, 안전재고, 바코드
- **연결고리**:
  - → 전표작성 (상품 선택)
  - → 재고관리 (수량 변동)
  - → 단가표 (가격 조회)

#### 3.1.4 자사은행계좌등록
- **목적**: 회사 보유 은행 계좌 관리
- **필드**: 은행명, 계좌번호, 예금주, 계좌별칭, 잔액
- **연결고리**:
  - → 수금/지급 전표 (입금 계좌 선택)
  - → 은행계좌명세서 (거래 내역)

#### 3.1.5 부서정보
- **목적**: 조직 구조 관리
- **필드**: 부서코드, 부서명, 상위부서
- **연결고리**:
  - → 사원등록 (소속 부서)
  - → 사원별 영업현황 (부서별 실적)

#### 3.1.6 부서별사원등록
- **목적**: 직원 정보 및 담당 업무 관리
- **필드**: 사원코드, 성명, 부서, 직급, 연락처, 이메일, 입사일
- **연결고리**:
  - → 거래처등록 (담당사원)
  - → 전표작성 (담당자)
  - → 사원별 영업현황

#### 3.1.7 입출금항목등록
- **목적**: 현금/은행 입출금 항목 분류
- **필드**: 항목코드, 항목명, 입금/출금 구분, 계정과목
- **연결고리**:
  - → 입출금 전표 (항목 선택)
  - → 비용수익 명세서 (항목별 집계)

#### 3.1.8 신용카드등록
- **목적**: 결제용 신용카드 정보 관리
- **필드**: 카드번호, 카드사, 유효기간, 한도액
- **연결고리**:
  - → 수금/지급 전표 (카드 결제)
  - → 매출카드명세서

#### 3.1.9 분류정보
- **목적**: 상품 분류 체계 관리
- **필드**: 분류코드, 분류명, 상위분류
- **연결고리**:
  - → 상품등록 (상품 분류)
  - → 상품별 매출/매입 현황 (분류별 필터)

#### 3.1.10 테마정보
- **목적**: 상품 테마별 그룹핑
- **필드**: 테마코드, 테마명, 설명
- **연결고리**:
  - → 상품등록 (테마 지정)
  - → 테마분류별 마진정보

#### 3.1.11 마진율그룹정보
- **목적**: 거래처별 적용 마진율 그룹 관리
- **필드**: 그룹코드, 그룹명, 기본마진율
- **연결고리**:
  - → 거래처등록 (마진그룹 지정)
  - → 자동 단가 계산

#### 3.1.12 상품분류별마진정보
- **목적**: 분류별 마진율 설정
- **필드**: 분류코드, 마진그룹, 마진율
- **연결고리**:
  - → 상품단가 자동 계산

#### 3.1.13 테마분류별마진정보
- **목적**: 테마별 마진율 설정
- **필드**: 테마코드, 마진그룹, 마진율
- **연결고리**:
  - → 테마별 단가 계산

#### 3.1.14 수당항목등록
- **목적**: 급여 수당 항목 관리
- **필드**: 항목코드, 항목명, 과세여부, 계산방식
- **연결고리**:
  - → 급여계산

#### 3.1.15 공제항목등록
- **목적**: 급여 공제 항목 관리
- **필드**: 항목코드, 항목명, 계산방식
- **연결고리**:
  - → 급여계산

### 3.2 기초이월 (Year-End Carry Forward)

#### 3.2.1 상품재고이월
- **목적**: 전년도 상품 재고를 신년도로 이월
- **필드**: 상품코드, 이월수량, 이월단가, 이월금액
- **연결고리**:
  - 상품등록 ← 기준 데이터
  - → 재고현황 (기초재고)

#### 3.2.2 은행잔고이월
- **목적**: 전년도 은행 잔액을 신년도로 이월
- **필드**: 계좌번호, 이월잔액
- **연결고리**:
  - 은행계좌등록 ← 기준
  - → 은행계좌잔고현황 (기초잔액)

#### 3.2.3 미수금이월
- **목적**: 전년도 미수금(외상매출금)을 신년도로 이월
- **필드**: 거래처코드, 이월미수금액
- **연결고리**:
  - 거래처등록 ← 기준
  - → 거래처별미수미지급현황 (기초미수)

#### 3.2.4 미지급금이월
- **목적**: 전년도 미지급금(외상매입금)을 신년도로 이월
- **필드**: 거래처코드, 이월미지급금액
- **연결고리**:
  - 거래처등록 ← 기준
  - → 거래처별미수미지급현황 (기초미지급)

#### 3.2.5 어음이월
- **목적**: 미결제 어음을 신년도로 이월
- **필드**: 어음번호, 거래처, 금액, 만기일
- **연결고리**:
  - → 어음명세서 (이월 어음)

#### 3.2.6 채권채무이월
- **목적**: 기타 채권/채무를 신년도로 이월
- **필드**: 거래처코드, 채권금액, 채무금액
- **연결고리**:
  - → 채권채무명세서

### 3.3 전표작성 (Voucher Entry)

#### 3.3.1 매출전표
- **목적**: 판매 거래 기록
- **필드**: 전표번호, 일자, 거래처, 상품목록(상품, 수량, 단가, 금액), 부가세, 합계, 담당자, 비고
- **연결고리**:
  - 거래처등록 → 거래처 선택
  - 상품등록 → 상품 선택
  - → 매출명세서 (조회)
  - → 미수금 증가 (외상 시)
  - → 재고 감소 (출고)
  - → 세금계산서 발행

#### 3.3.2 매입전표
- **목적**: 구매 거래 기록
- **필드**: 전표번호, 일자, 거래처, 상품목록, 부가세, 합계, 담당자, 비고
- **연결고리**:
  - → 매입명세서
  - → 미지급금 증가 (외상 시)
  - → 재고 증가 (입고)

#### 3.3.3 수금전표
- **목적**: 매출대금 회수 기록
- **필드**: 전표번호, 일자, 거래처, 수금방법(현금/카드/어음/계좌이체), 금액, 입금계좌
- **연결고리**:
  - → 미수금 감소
  - → 현금/은행 잔액 증가
  - → 수금명세서

#### 3.3.4 지급전표
- **목적**: 매입대금 지급 기록
- **필드**: 전표번호, 일자, 거래처, 지급방법, 금액, 출금계좌
- **연결고리**:
  - → 미지급금 감소
  - → 현금/은행 잔액 감소
  - → 지급명세서

#### 3.3.5 입출금전표
- **목적**: 현금/은행 입출금 기록 (거래처 무관)
- **필드**: 전표번호, 일자, 입출금구분, 항목, 금액, 계좌, 적요
- **연결고리**:
  - → 현금/은행 잔액 변동
  - → 비용수익명세서

#### 3.3.6 입고전표
- **목적**: 재고 입고 기록 (매입 외 입고)
- **필드**: 전표번호, 일자, 상품, 수량, 단가, 창고, 비고
- **연결고리**:
  - → 재고 증가
  - → 입고명세서

#### 3.3.7 출고전표
- **목적**: 재고 출고 기록 (매출 외 출고)
- **필드**: 전표번호, 일자, 상품, 수량, 창고, 출고사유
- **연결고리**:
  - → 재고 감소
  - → 출고명세서

#### 3.3.8 비용수익전표
- **목적**: 기타 비용/수익 기록
- **필드**: 전표번호, 일자, 구분(비용/수익), 항목, 금액, 적요
- **연결고리**:
  - → 비용수익명세서
  - → 손익현황

#### 3.3.9 채권채무전표
- **목적**: 기타 채권/채무 발생 기록
- **필드**: 전표번호, 일자, 거래처, 구분(채권/채무), 금액, 사유
- **연결고리**:
  - → 채권채무명세서

### 3.4 매출관리 (Sales Management)

#### 3.4.1 매출명세서
- **목적**: 매출 거래 상세 조회
- **조회조건**: 기간, 거래처, 상품, 담당자
- **출력항목**: 일자, 전표번호, 거래처, 상품, 수량, 단가, 공급가, 부가세, 합계
- **연결고리**: 매출전표 데이터

#### 3.4.2 매출처원장_A (상세형)
- **목적**: 거래처별 매출 거래 상세 내역
- **출력항목**: 일자, 상품, 수량, 단가, 금액, 수금, 잔액

#### 3.4.3 매출처원장_B (요약형)
- **목적**: 거래처별 매출/수금 잔액 추이
- **출력항목**: 일자, 매출, 수금, 잔액

#### 3.4.4 거래처별매출현황
- **목적**: 거래처별 매출 합계 분석
- **출력항목**: 거래처, 매출건수, 매출금액, 부가세, 합계

#### 3.4.5 상품별매출현황
- **목적**: 상품별 매출 분석
- **출력항목**: 상품, 판매수량, 매출금액, 원가, 마진

#### 3.4.6 상품별매출집계표
- **목적**: 상품별 월별/분기별 매출 집계
- **출력항목**: 상품, 1월~12월, 분기별, 연합계

#### 3.4.7 상품별매출집계표_상세
- **목적**: 상품별 일별 매출 상세
- **출력항목**: 일자, 거래처, 수량, 금액

#### 3.4.8 업체별매출집계표
- **목적**: 거래처별 월별 매출 집계
- **출력항목**: 거래처, 월별금액, 합계

#### 3.4.9 업체별매출집계표_상세
- **목적**: 거래처별 일별 매출 상세

### 3.5 매입관리 (Purchase Management)

#### 3.5.1 매입명세서
- **목적**: 매입 거래 상세 조회
- **출력항목**: 일자, 거래처, 상품, 수량, 단가, 금액

#### 3.5.2 매입처원장
- **목적**: 거래처별 매입/지급 내역

## 4. 데이터 흐름도 (Data Flow)

### 4.1 매출 프로세스
```
견적서관리 ──→ 매출전표 ──→ 재고 감소
     │              │              │
     │              ├──→ 미수금 증가
     │              │
     │              └──→ 세금계산서 발행
     │
     └──────────────────────────────→ 매출명세서/현황
```

### 4.2 매입 프로세스
```
발주서관리 ──→ 매입전표 ──→ 재고 증가
     │              │              │
     │              ├──→ 미지급금 증가
     │              │
     │              └──→ 세금계산서 등록
     │
     └──────────────────────────────→ 매입명세서/현황
```

### 4.3 수금/지급 프로세스
```
미수금 ──→ 수금전표 ──→ 미수금 감소
                │              │
                └──→ 현금/은행 증가

미지급금 ──→ 지급전표 ──→ 미지급금 감소
                 │              │
                 └──→ 현금/은행 감소
```

### 4.4 재고 프로세스
```
기초이월 ──→ 재고현황 ←── 매입전표(입고)
     │           │
     │           ├── 매출전표(출고)
     │           │
     │           ├── 입고전표(직접입고)
     │           │
     │           └── 출고전표(직접출고)
     │
     └──→ 재고조정 ──→ 재고현황 보정
```
